Create these under a seperate JMS Module with its own JMS Server manually.
Don't use the copy paste method mentioned in HTML document.

If we start adding Live external systems to replace the emulator then please create new request/response queues with same names by replacing ExternalSystemEmulator with Actual system name. 
Add duplicate automation tasks and use some keyword in RequestXML like EMULATOR/LIVE to route to live or emulator automation task.
You can also add a manual task before each system interaction task to achieve the same.
