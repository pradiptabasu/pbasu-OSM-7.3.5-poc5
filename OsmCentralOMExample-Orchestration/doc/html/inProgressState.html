In Progress Order State

The order is being provisioned. There is at least one live task running on the order.

Transitions

Abort Order:

Cancel Order: Rolls back order to the creation task. You can specify a grace period to wait for all accepted tasks to complete before transitioning the order.

Fail Order:

Process Amendment: Initiates a transition to the Amending order state.

Raise Exception: Permits process exceptions to be raised.

Submit Amendment: Submits an amendment (typically from an external CRM system) and is followed by Process Amendment.

Suspend Order: Halt processing of an order. You can specify a grace period to wait for all accepted tasks to complete before transitioning the order.

Update Order: Update the data on an order.
